# Ledebe — Pricing & Tiers

## Guiding Principle
Free tier should be genuinely useful — enough to hook users and prove value.
Paid tier should unlock features that teams and power users actually need.

---

## Recommended Tier Structure

### Free Tier — "Personal"
**Price:** £0/month forever
**Who it's for:** Individuals trying it out, light users, developers

**What's included:**
- Unlimited text protection (core feature — never limit this)
- Up to 20 custom terms
- File upload — up to 3 files per day (PDF, DOCX, images)
- Chat history — last 30 conversations
- VS Code extension — full access
- Web app access
- Dark/light mode
- All keyboard shortcuts

**What's NOT included:**
- AI chat (Ask AI mode) — requires own API key at free tier
- Company sync
- Priority support
- Advanced reporting

---

### Pro Tier — "Professional"
**Price:** £8/month (or £75/year — save 20%)
**Who it's for:** Individual professionals, freelancers, power users

**What's included (everything in Free, plus):**
- Unlimited custom terms
- Unlimited file uploads (up to 50MB per file)
- AI chat with own API key + usage tracking
- Full chat history (unlimited)
- Export protected documents
- Priority email support
- Early access to new features

---

### Team Tier — "Business"
**Price:** £18/user/month (minimum 3 users, or £160/user/year)
**Who it's for:** SMBs, teams, departments

**What's included (everything in Pro, plus):**
- Company sync — shared custom terms across the whole team
- Admin dashboard — see team usage, manage members
- Centrally managed protected terms (IT/admin controls what gets masked)
- Audit log — who protected what, when
- SSO (Single Sign-On) — coming soon
- Dedicated account manager (10+ users)
- Invoice billing

---

### Enterprise — "Custom"
**Price:** Custom quote (typically £30-50/user/month, annual contract)
**Who it's for:** Large organisations, regulated industries

**What's included (everything in Team, plus):**
- On-premise deployment option
- Custom integrations
- SLA guarantee (99.9% uptime)
- Compliance reports (GDPR, HIPAA)
- Custom term library (pre-loaded with industry-specific sensitive terms)
- Dedicated support and onboarding
- Volume discounts

---

## What Should NEVER Be Behind a Paywall
- Core PII masking — this is the product, locking it kills trust
- Restore original — users must always be able to get their data back
- VS Code extension basic features — acquisition channel, keep it free

---

## Competitor Pricing Reference
- Most enterprise data loss prevention (DLP) tools: £20-100/user/month
- Ledebe undercuts significantly while offering a simpler, faster experience
