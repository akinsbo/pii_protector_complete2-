# v1.2.1

Release date: 2026-06-28

## Highlights

- Added selection-based placeholder toggle in the browser extension.
- Selecting a placeholder token and using the Ledebe option now reveals the real value in place.
- Works for both auto-detected placeholders and custom-word placeholders.
- Kept the left drawer toggle behavior covered by Playwright regression tests.

## Verification

- `node --check ledebe-browser-extension/content.js`
- `node --check ledebe-browser-extension/background.js`
- `npx playwright test -c ledebe-browser-extension/e2e/playwright.config.ts ledebe-browser-extension/e2e/panel.spec.ts`

## Notes

- The extension UI internationalization is in place through `ledebe-browser-extension/i18n.js`.
- Website internationalization is in place through `pii_protector_website/i18n.js`.
- Chrome extension `_locales` resources are not yet present, so the browser-native context-menu title is still English.
- The legacy jsdom DOM harness remains blocked by the existing `html-encoding-sniffer` / ESM dependency issue in this repo.
