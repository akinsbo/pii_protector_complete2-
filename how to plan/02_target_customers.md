# Ledebe — Target Customers

## Segment 1: Individual Professionals (B2C)
**Who:** Lawyers, accountants, HR managers, consultants, healthcare workers
**Problem:** They use AI to draft documents, summarise reports, write emails — often pasting real client data
**Pain level:** High — GDPR/HIPAA violations can result in heavy fines
**Willingness to pay:** Medium — £5-10/month personally, more if company pays
**How to reach:** LinkedIn, professional forums, word of mouth

### Examples:
- A lawyer drafting a contract and asking ChatGPT to improve the language — client names, case numbers exposed
- An HR manager asking AI to summarise a performance review — employee data exposed
- A consultant pasting a client report into Claude for analysis — confidential business data exposed

---

## Segment 2: Developers (B2C / B2B)
**Who:** Software engineers, DevOps, data engineers
**Problem:** They paste code with API keys, DB credentials, `.env` files, logs with real user data into AI coding assistants
**Pain level:** Very high — API key leaks cause real financial damage
**Willingness to pay:** Medium — prefer free tools but will pay for good ones
**How to reach:** VS Code Marketplace, Hacker News, Reddit (r/devops, r/programming), GitHub

### Examples:
- Developer pastes a `.env` file into ChatGPT asking for help — AWS keys exposed
- Data engineer shares a CSV sample with real customer emails asking for a transformation script
- DevOps engineer pastes a Kubernetes config with internal IPs and credentials

---

## Segment 3: Small & Medium Businesses (B2B)
**Who:** Companies with 10-500 employees adopting AI tools across teams
**Problem:** Staff are using AI without any data governance — no visibility, no control
**Pain level:** Very high — one breach can cost millions in GDPR fines
**Willingness to pay:** High — £20-50/user/month, or flat team pricing
**How to reach:** Cybersecurity consultants, LinkedIn outreach to CTOs/DPOs, compliance events

### Examples:
- A marketing team pasting customer lists into AI for segmentation
- A sales team using AI to draft emails with real prospect data
- Customer support using AI to summarise tickets containing personal information

---

## Segment 4: Enterprise / Compliance-Driven Organisations (B2B)
**Who:** Banks, insurance companies, NHS, law firms, any regulated industry
**Problem:** Strict regulatory requirements around data handling — AI adoption is happening anyway
**Pain level:** Extreme — regulatory fines, reputational damage
**Willingness to pay:** Very high — £50-200/user/month, annual contracts
**How to reach:** Direct sales, cybersecurity events, partnerships with compliance consultants

---

## Best Starting Segment (Now)
**Developers via VS Code extension** — lowest friction, self-serve, spreads organically
**Individual professionals** — easiest to reach via LinkedIn content

## Best Revenue Segment (6-12 months)
**SMBs** — team plans, higher revenue per customer, referral within organisations
