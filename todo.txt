==High Priority (Broken/Missing)
Help button does nothing useful — clicking ❓ Help just shows keyboard shortcuts. Should open actual help content or the docs page.
AI provider checkboxes (ChatGPT/Claude/Gemini) in sidebar — the checkboxes exist visually but only one provider can actually be active. Selecting one doesn't properly switch and disable others in real-time.
Notifications setting does nothing — the "Enable notifications" checkbox in Settings is saved but never actually triggers any OS notifications.
No loading indicator when AI is thinking — user gets no visual feedback while waiting for an AI response. The send button just says "Sending..." but there's no spinner or progress state.
PII data logged to browser console — placeholderMap (with real names/emails) is logged to the console on restore. A user could accidentally expose their data if they share a screenshot with DevTools open.

🟡 Medium Priority (UX & Polish)
No timestamps on chat messages — you can't tell when a message was sent. History items show dates but individual chat bubbles don't.
No file size warning on upload — uploading a very large PDF or image gives no warning; OCR can hang silently for 30+ seconds.
Empty history sidebar is too plain — just says "No history yet". Could guide a new user on what to do.
No confirmation before clearing custom terms — if a "clear all" action ever fires, all terms vanish with no warning.
Dark mode contrast — a few elements (some form inputs and modal borders) don't have enough contrast in dark mode.
Max 20 history items — hardcoded limit with no way for the user to increase it or export older items.
No search/filter for history — as history grows, there's no way to find a past entry without scrolling.
Language translation only covers some UI strings — header, sidebar, and settings are translated, but AI response labels, toast messages, and error text are still English-only.

🟢 Nice to Have (Future Features)
Clipboard auto-protect — automatically detect and mask PII when the user pastes from clipboard, without having to click Send first.
Custom regex PII patterns — let power users define patterns like INV-\d{5} or EMP-\w+ beyond just word-based custom terms.
Export/import history — download history as JSON or CSV, and re-import it on another device.
Message timestamps visible in chat — small grey timestamp under each bubble.
Batch file upload — process multiple files at once instead of one at a time.
Undo for message edits — once you save an edited message, the original is gone.
Analytics summary — a small "this month you protected 47 emails, 12 phone numbers" stat somewhere in the sidebar.
