# Multi-AI Quick Reference Card

## 🚀 Quick Start (2 Minutes)

### Step 1: Get API Keys
```
OpenAI:     https://platform.openai.com/api-keys
Anthropic:  https://console.anthropic.com/account/keys
Google AI:  https://aistudio.google.com/app/apikey
```

### Step 2: Configure
1. Open AI Chat
2. Click "Configure API Keys"
3. Paste your keys
4. Keys auto-save ✅

### Step 3: Enable & Chat
1. Check boxes for AIs you want
2. Type your message
3. Get multiple responses!

## 📋 Checkbox Interface

```
☑ ChatGPT (OpenAI)      ✅  ← Enabled, key configured
☑ Claude (Anthropic)    ✅  ← Enabled, key configured
☐ Gemini (Google)       ⚠️  ← Disabled, needs key
```

**Icons:**
- ✅ = API key configured
- ⚠️ = API key required
- ☑ = Enabled
- ☐ = Disabled

## 🐛 Troubleshooting

### Checkbox Disabled
**Problem:** Can't check the box
**Solution:** Configure API key first

### No Response
**Problem:** AI not responding
**Solution:** Check API key validity

### Error Message
**Problem:** Red error box appears
**Solution:** Click "Get API Key" link

### Rate Limit
**Problem:** "Rate limit exceeded"
**Solution:** Wait or use different AI

## 🔧 Settings

Each AI has:
- **API Key** (required)
- **Model** (select from dropdown)
- **Temperature** (0.0-1.0, creativity)
- **Max Tokens** (response length)

## 📱 Tips

## 🆘 Quick Help

**Can't enable AI?**
→ Add API key first

**Multiple errors?**
→ Check internet connection

**Responses too short?**
→ Increase Max Tokens

**Responses too random?**
→ Lower Temperature

**Too expensive?**
→ Use Gemini Flash or GPT-3.5
