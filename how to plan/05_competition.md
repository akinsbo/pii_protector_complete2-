# Ledebe — Competition Analysis

## Direct Competitors

### 1. Enterprise DLP Tools (Symantec, Forcepoint, Microsoft Purview)
**What they do:** Monitor and block data leaving an organisation — email, USB, web traffic
**Price:** £20-100+/user/month, annual contracts, complex setup
**Weakness:** Built for IT departments, not individual users. Too complex, too expensive for SMBs. Don't specifically address AI tools.
**Ledebe advantage:** Simple, instant, works without IT involvement, purpose-built for AI

### 2. ChatGPT Enterprise / Claude for Work
**What they do:** Enterprise versions of AI with data privacy guarantees from the provider
**Price:** £20-60/user/month
**Weakness:** Locks you into one AI provider. Doesn't help if staff use personal ChatGPT accounts. Doesn't work across multiple AI tools.
**Ledebe advantage:** Works with ANY AI tool — ChatGPT, Claude, Gemini, Copilot, all of them

### 3. PrivateAI
**What they do:** API-based PII detection and masking — aimed at developers building AI pipelines
**Price:** Usage-based API pricing
**Weakness:** Requires technical integration — not for end users. No consumer product.
**Ledebe advantage:** No integration needed — works as a standalone tool anyone can use

### 4. Nightfall AI
**What they do:** DLP for cloud apps — Slack, Google Drive, GitHub
**Price:** Enterprise pricing
**Weakness:** Focused on cloud storage, not AI prompts specifically
**Ledebe advantage:** Purpose-built for AI prompt protection

## Indirect Competitors

### Browser extensions that warn about PII
- Basic, no masking capability, no restore feature
- Ledebe is significantly more capable

### Copy-paste habit / manual editing
- Most people's current "solution" — manually delete sensitive bits before pasting
- Slow, error-prone, no guarantee everything is caught
- Ledebe automates this completely
