# Ledebe — Business Overview

## What Ledebe Is
A privacy tool that masks sensitive personal data (PII) before it reaches AI tools like ChatGPT, Claude, and Copilot. It works locally — no data leaves the user's device during protection.

## The Problem It Solves
Every day, millions of people paste emails, phone numbers, API keys, client records, and contracts into AI tools. Most don't realise that:
- AI providers may use prompts for model training
- Companies can violate GDPR/HIPAA by sharing customer data with third-party AI
- Developers regularly leak API keys and credentials into AI assistants

Ledebe sits between the user and the AI — masking the sensitive data before it leaves.

## What Makes It Different
- Works locally — PII detection happens in the browser/app, not on a server
- Reversible — placeholders can be restored to original values
- Flexible — users can add custom terms (names, company names, project codes)
- Multi-platform — web app, desktop app (Mac/Windows/Linux), VS Code extension
- Multi-AI — works with any AI tool, not locked to one provider

## Vision
Become the standard privacy layer that sits between humans and AI — used by individuals, developers, and enterprises to ensure sensitive data never reaches AI systems unprotected.
